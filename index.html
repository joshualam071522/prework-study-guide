HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • You must link the CSS file to the html file to apply the style by using the link element in the section

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name:creates a new branch and switches to it
  • git add -A: Stages all the files so all saved changes are recorded to Git
  • git commit -m "commit-message": Commits changes
  • git pull origin main: Ensures main branch is up to date on all changes
  • git push origin branch-name: Creates pull request for changes on the requested branch

JavaScript

  • A Variable is a nemd container that allows us to store data in our code
  • Control flow is the order in which a computer executes code in a script
  • Read top to bottom
  • declare a function first, then call the function
